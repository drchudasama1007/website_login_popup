Website Login/Signup Popup

This module allows user can Login/Signup By Popup From Any Page of Website. This module has Login/Signup Functnality By Ajax with all validation.

Enable Signup Option (Website > Configuation > Free sign up)

Enable Password Reset (Settings > General Settings > Password Reset)

We open Login/Signup Popup in any page of website

Login Popup - Login Functnality and Validations By Ajax

Signup Popup - Signup Functnality and Validations By Ajax

User Dashboard after successfully signup From Popup
